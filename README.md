# Machine Learning Portfolio 🚀

Welcome to my portfolio! I am Elif Er, a senior Mathematics student at Istanbul Medeniyet University, passionate about applying mathematical foundations to Machine Learning and Data Science problems.

This repository contains end-to-end Machine Learning projects focusing on classification and regression tasks. 

## 1. End-to-End Multi-Class Classification Pipeline 🔩
**File:** `steel_plates_pipeline.ipynb`

A comprehensive classification pipeline developed to predict steel plate surface faults using the Steel Plates Faults Dataset (UCI). 
* **Workflow:** Exploratory Data Analysis (EDA), Preprocessing, SMOTE comparison, Feature Selection (3 methods), Dimensionality Reduction (PCA), and Model Evaluation.
* **Algorithms Evaluated:** XGBoost, LightGBM, Support Vector Machines (LinearSVC, SVC), Random Forest, Gradient Boosting, K-Nearest Neighbors, and Logistic Regression.
* **Outlier Detection:** Local Outlier Factor (LOF) analysis applied for identifying anomalies within the dataset.

## 2. Sleep Efficiency Regression Analysis 🛏️
**File:** `uyku_kalitesi.py`

A regression analysis project aiming to predict sleep efficiency based on various lifestyle and demographic factors.
* **Data Processing:** Missing value imputation achieved using KNNImputer, with outlier handling managed via IQR clipping and RobustScaler.
* **Algorithms Used:** Ordinary Least Squares (OLS) Linear Regression, Lasso Regression (with Cross-Validation), and RANSAC Regressor for robust management of outliers.
* **Evaluation:** Model performances compared utilizing Mean Squared Error (MSE), Mean Absolute Error (MAE), and R-Squared (R2) metrics.
